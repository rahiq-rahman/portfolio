# Rahiq Rahman — Portfolio

A modern, aesthetic portfolio built with **Vite + React**.

## Setup

```bash
# 1. Install dependencies
npm install

# 2. Start dev server
npm run dev

# 3. Open http://localhost:5173
```

## Build for production

```bash
npm run build
npm run preview
```

## Features
- Custom animated cursor (dot + ring)
- Smooth scroll-reveal animations
- 3D tilt cards on hover
- Animated marquee ticker
- Fully responsive (mobile-first)
- Noise texture overlay
- Gold/warm ink aesthetic with Cormorant Garamond + Syne fonts
